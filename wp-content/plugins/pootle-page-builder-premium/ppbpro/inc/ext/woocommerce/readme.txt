== Description ==

pootle page builder for WooCommerce brings powerful WooCommerce features into page builder.

With pootle page builder for WooCommerce you can:

 * Create stunning pages featuring your products by id, category, attribute, best-selling, top rated and on-sale
 * Use pootle page builder on product pages
 * Use pootle page builder with WooCommerce Tab Manager

 Yes! Watch this video.
[vimeo http://vimeo.com/137234634]

== Usage ==

Install and activate the plugin. In your WordPress dashboard simply add a new page to start using with page builder.

== Installation ==

Installing "pootle page builder for WooCommerce" can be done by using the following steps:

1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy pootle page builder with WooCommerce
